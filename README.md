# Certificate Archive

Next.js App Router portfolio page for displaying Coursera certificates, field grouping, modal previews, and chart-based data views.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

Import this folder as a Vercel project. The default build command is `npm run build`; Vercel will detect Next.js automatically.
